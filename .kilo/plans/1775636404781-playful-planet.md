# Plan de Implementación: v0.4.0 - SHARING + ANALYTICS

## Objetivo

Implementar funcionalidad para compartir perfiles y métricas básicas (QR, vCard, Analytics)

## Estado Actual

- ✅ v0.1.0 - Foundation (auth, Next.js setup, Prisma)
- ✅ v0.2.0 - Core Links (CRUD completo de links)
- ✅ v0.3.0 - Customization (perfiles, temas) - PARCIAL

## Estado: v0.3.0 Customization (pendiente)

El schema de Prisma ya tiene Profile con todos los campos necesarios, pero faltan:

- Endpoints API: `/api/profiles/me` (GET/PUT), `/api/themes`
- Frontend: ProfileEditor, ThemeSelector, PreviewModal
- Tema personalizado aplicado en `/[username]`

## Siguiente: v0.4.0 - SHARING + ANALYTICS

### Backend

| ID  | Endpoint                          | Método                        | Descripción |
| --- | --------------------------------- | ----------------------------- | ----------- |
| Q1  | GET /api/profiles/:username/qr    | Generar QR con URL del perfil |             |
| V1  | GET /api/profiles/:username/vcard | Generar archivo .vcf          |             |
| A1  | POST /api/analytics/click         | Registrar click en link       |             |
| A2  | POST /api/analytics/visit         | Registrar visita al perfil    |             |
| A3  | GET /api/analytics/summary        | Métricas del dashboard        |             |

### Dependencias

- `npm install qrcode` - para generar QR codes
- `npm install uuid` - para visitor IDs

### Frontend

| ID  | Componente         | Descripción                               |
| --- | ------------------ | ----------------------------------------- |
| C1  | QRCodeModal        | Generar y descargar QR                    |
| C2  | vCardButton        | Descargar contacto                        |
| C3  | AnalyticsDashboard | Stats overview                            |
| C4  | ClickChart         | Gráfico de clicks (recharts ya instalado) |
| C5  | TopLinksTable      | Links más populares                       |

### Flujo de Implementación

#### Fase 1: Completar v0.3.0 (Customization)

1. Crear `/api/profiles/me` GET/PUT
2. Crear ProfileEditor component
3. Crear ThemeSelector component
4. Actualizar `/[username]` para usar theme/backgroundColor
5. Agregar página Settings/Profile

#### Fase 2: QR Code

1. Instalar `qrcode`
2. Crear GET `/api/profiles/:username/qr`
3. Retornar PNG base64
4. Crear QRCodeModal component

#### Fase 3: vCard

1. Crear GET `/api/profiles/:username/vcard`
2. Retornar archivo .vcf con FN, EMAIL, URL
3. Crear vCardButton component

#### Fase 4: Analytics

1. Crear POST `/api/analytics/click`
2. Crear POST `/api/analytics/visit`
3. Crear GET `/api/analytics/summary`
4. Actualizar `/[username]` para llamar visit endpoint
5. Actualizar links para llamar click endpoint
6. Crear AnalyticsDashboard component
7. Crear ClickChart component (usar Recharts)
8. Crear TopLinksTable component

### Definition of Done

- [ ] QR code genera y descarga correctamente
- [ ] vCard se importa en contactos
- [ ] Dashboard muestra métricas reales
- [ ] Clicks se registran al hacer click
